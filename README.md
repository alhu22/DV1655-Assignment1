# Simple Compiler Project

## 📌 Introduction
This is a simple compiler project designed to read Java source files, perform lexical, syntactic, and semantic analysis, and generate corresponding bytecode.

## 🚀 Requirements
- **Operating System:** Unix-based (Linux or macOS)
- **Dependencies:** 
  - GCC (GNU Compiler Collection)
  - Make utility

## ⚡️ Setup and Compilation
1. **Clone the repository** or download the source files.
2. **Open a terminal** in the project directory.
3. **Compile the project** using the command:
   ```bash
   make
